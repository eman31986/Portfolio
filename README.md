## 📬 Contact Form Setup

The contact form sends data directly to **Google Sheets** via Google Apps Script.

**Flow:**
```
User fills form → Vanilla JS fetch() → Apps Script Web App → Google Sheets ✅
```

To set up your own:
1. Create a Google Sheet
2. Go to **Extensions → Apps Script** and paste the `doPost` function
3. Deploy as **Web App** (Anyone can access)
4. Replace `APPS_SCRIPT_URL` in the HTML with your deployment URL
